# Java-Contacts-Exercice

The project aims to help learners practice their Java skills by creating a real-world application from scratch. The project consists of several stages, each with a set of requirements and tests.

## How to Run it

1. Clone Repository `git clone https://github.com/JoaoPedroPereiraOliveira/Java-Contacts-Exercice.git`

2. In Terminal go to ditectory and go `cd .\contants`

3. Run `java -cp . contacts.Main`

## How it works

Here is something you probably use regularly: contacts. In this project, is a program to create contacts (like on the mobile phone) and search for people or organizations by name.

1. Files
   - Main.java
        * Run the project
   - Contacts.java
        * Contacts object
   - ContactList.java
        * Craetes Contacts List
   - Organization.java
        * Organization object
   - Person.java
        * Person object
   - PhoneBook.java
        * Stores Contacts
   - Record.java
        * Manipulates Contacts
   - Search.java
        * Finds Contacts
   - Menu.java
        * Main menu of the progarm
        * Calls Actions
   - Actions.java
        * Sends and resives Request for the user
   - ValidInput.java
        * Validation for input of the user
        * Verify Birt Date
        * Verify Gender
        * Verify Birt Date
        * Verify Phone Number
        * If is a valid Contact
        
2. Features
   - Add Contact
        * Type Person / Organization
        * Name
        * Address
        * Birth date
        * Gender
        * Phone Number
        * Create specific object Person / Organization whit diffrent requests
   - List
        * List all contacts
        * Used also to edit and delete
   - Search
        * Find contact by Search
        * Used also to edit and delete
   - count
        * Count ammout of contacts
   - Exit
        * End the program
